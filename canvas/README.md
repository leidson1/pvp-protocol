# PVP Canvas — Instrumentos de Registro

Três versões do instrumento de registro pós-aula do PVP.

| Arquivo | Formato | Uso |
|---------|---------|-----|
| `pvp-canvas-completo.html` | A4 paisagem, colorido | Canvas + Guia Rápido (2 páginas) |
| `pvp-canvas-monocromatico.html` | A4 paisagem, P&B | Impressão econômica (2 páginas) |
| `pvp-canvas-mini.html` | A5 paisagem, P&B | Versão de bolso para anexar (1 página) |

## Como imprimir

Abra o arquivo `.html` no navegador e use `Ctrl+P` (ou `Cmd+P`).
Configure para paisagem e margens mínimas.

O Canvas Mini (A5) imprime dois por folha A4 se configurar "2 páginas por folha".

## Como usar

1. **Antes da aula:** preencha a primeira coluna
2. **Depois da aula:** preencha a segunda e terceira colunas
3. **Registre:** uma linha no histórico de versões
4. **Anexe:** ao plano de aula correspondente

Tempo estimado: 2 minutos após a aula.
